Le jeu du Pendu

Description

Le but du jeu est simple : deviner toutes les lettres qui composent un mot choisi aléatoirement par l'ordinateur, avec un nombre limite de tentatives fixé à 7. A chaque fois que le joueur devine une lettre, celle-ci est affichée. Dans le cas contraire, le dessin d’un pendu se met à apparaître…

4.8

Configuration minimale :

Processeur Intel Core I3-4130 / AMD Athlon II X2 240
Carte Graphique Intel HD Graphics 4400 / NVIDIA GeForce GTX 660
Mémoire Vive 4Go de RAM
Stockage 30Mo de stockage
OS Windows 10
