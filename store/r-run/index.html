R-Run

Description

Vous connaissez sûrement le jeu du dinosaure sur Google Chrome... Et bien R-Run est une version améliorée de ce petit jeu ! Dans une ambiance de jeu 8-bit, courez le plus longtemps sans vous faire arrêter par un ennemi ! Le score moyen des joueurs est de 62, arriverez vous à faire mieux ?

4.9
